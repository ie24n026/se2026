# 映画管理CLI

見た映画を登録して、評価や感想を管理するCLIソフト。

## 機能要件
1. 映画を追加する
      - 映画タイトルを指定して追加
      - ジャンルを指定して追加
      - 評価を指定して追加
      - 感想を指定して追加
2. 登録済みの映画を編集する
3. 映画を削除する
4. 映画の一覧を表示する
5. 映画名で検索する
6. 登録した映画の統計を表示する

## 非機能要件
1. CLIで動く
2. 内部データはCSVで保存
3. 日本語が文字化けしない（UTF-8)
4. Python標準ライブラリだけで動く
5. 各コマンドの処理は1秒以内
